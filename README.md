# 🧠 Oracle – AI Voice Assistant (Web-Based)

## 🚀 Features

- 🎙️ Voice recognition (click-to-talk)
- 🗣️ AI speech response
- 🔍 Opens sites like Google, YouTube, Wikipedia, and more

- 🧠 Smart search for "what is", "who is", etc.
